# LUMERYA CORE 🔧
_Plugin principal du serveur survival Lumerya_

## Description 🧾
Lumerya Core regroupe les systèmes fondamentaux du serveur Minecraft :
* Gestion des joueurs
* Commandes principales
* Systèmes utilitaires
* API interne pour les autres plugins
* Optimisations et outils serveur

## Serveurs compatibles 🧩
* Spigot
* Paper
* Bukkit
